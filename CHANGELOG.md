# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-06-12

### Added
- Initial release
- Complex binary occupancy grid map setup using MATLAB `exampleMaps`
- Obstacle inflation with configurable robot radius
- PRM (Probabilistic Roadmap) path planner
- Waypoint trajectory with heading angle computation
- Pure Pursuit controller for differential drive robot simulation
- Video recording of simulation (`robot_simulation.avi`)
- RRT (Rapidly-exploring Random Tree) path planner
- RRT* (optimal RRT) path planner
- Final analysis: path efficiency, position error, travel metrics
- Output plots: position profile, heading angle, distance to goal, planned vs actual path
